## 실험 데이터 복구하기

진호가 데이터를 날려 먹어서 데이터를 위조하려고 힌다. 데이터의 조각을 입력하면 조각을 모두 포함하는 가장짧은 문자열을 반환 하는 문제.

> 입력

```
3 <- 테스트케이스의 수
3 <- 데이터 조각 수
geo
oji
jing
2
world
hello
3
abrac
cadabra
dabe
```

>출력

```
geojing
helloworld
cadabrac
```
